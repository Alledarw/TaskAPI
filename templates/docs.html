API Documentation

Get all tasks

Endpoint: GET /tasks

Description: Get a list of all tasks.

Get specific task

Endpoint: GET /tasks/{task_id}

Description: Get a specifik task by id.

Get all categories

Endpoint: GET /categories

Description: Get a list of all categories.

Get specific categories

Endpoint: GET /categories/category

Description: Get specific category by name.

Create a new task

Endpoint: POST /tasks

Description: Create a new task.

Request JSON:

    {
        "id": int,
        "title": str,
        "description": str
    }

Delete a specific task

Endpoint: DELETE /tasks/{task_id}

Description: Delete a specifik task by id.

Request Headers: 

    {
        "Key": X-Secret-key,
        "Value": Testkey,
    }

Update specific task

Endpoint: PUT /tasks/{task_id}

Description: Update a specifik task by id.

Request JSON: (optional to use all keys)

    {
        "id": int,
        "title": str,
        "description": str
    }

Change status of a task

Endpoint: PUT /tasks/{task_id}/complete

Description: Change status of a specific task by id.

Add new task via frontend

Endpoint: GET /add

Description: Add a new task via the form.
