API Documentation

Get all tasks

Endpoint: GET /tasks

Description: Get a list of all tasks.

Get specific task

Endpoint: GET /tasks/{task_id}

Description: Get a specific task by id.

Get all categories

Endpoint: GET /categories

Description: Get a list of all categories.

Get specific category

Endpoint: GET /categories/category

Description: Get a specific category by name.

Create a new task

Endpoint: POST /tasks

Description: Create a new task.

Request JSON:

        {
            "id": int,
            "title": str,
            "description": str
        }

Delete a specific task

Endpoint: DELETE /tasks/{task_id}

Description: Delete a specific task by id.

Request Headers:

        {
            "Key": X-Secret-key,
            "Value": Testkey
        }

Update specific task

Endpoint: PUT /tasks/{task_id}

Description: Update a specific task by id.

Request JSON: (optional to use all keys)

        {
            "id": int,
            "title": str,
            "description": str
        }

Change status of a task to complete

Endpoint: PUT /tasks/{task_id}/complete

Description: Change the status of a specific task by id.

Change status of a task to incomplete

Endpoint: PUT /tasks/{task_id}/incomplete

Description: Change the status of a specific task by id.

Add a new task via frontend

Endpoint: GET /add

Description: Add a new task via the form.

Add new key (important)

Endpoint: PUT /tasks/{task_id}/important

Description: Adds a new key value "important".

Request JSON: (need to use "value" as a key)

        {
            "value": "important"
        }

Run test_app.py

Endpoint: pytest test_app.py

Description: Perfom tests on the API app. Run it with terminal
